## Empfohlene Bildspezifikationen:
- **Format**: PNG oder JPG
- **Auflösung**: 800x600px oder ähnliches 4:3 Format
- **Dateigröße**: Unter 500KB pro Bild (für schnelle Ladezeiten)
- **Qualität**: Hochauflösend, gut lesbare Screenshots

## Hinweise:
- Die Screenshots sollten echte Kundenergebnisse zeigen
- Sensible Daten sollten anonymisiert werden
- Die Bilder werden automatisch responsive skaliert
